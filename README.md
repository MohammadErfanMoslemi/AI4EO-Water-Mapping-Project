# Surface Water Segmentation from Sentinel-2: Spectral Baselines, Random Forest, and U-Net on Sen1Floods11

This repository contains the final project for an AI / Machine Learning for Earth Observation course. The project benchmarks surface-water segmentation methods on the hand-labeled Sentinel-2 subset of Sen1Floods11 and compares:

- NDWI
- MNDWI
- 13-band Random Forest
- 13-band U-Net 

The motivating downstream application is reservoir monitoring in the Tehran region, but the final submission focuses on a controlled benchmark study on Sen1Floods11.

## Project summary

The goal of this project is to compare simple spectral water indices, pixel-wise multispectral machine learning, and spatial deep learning under a common Sentinel-2 benchmark.

A key motivation of the final report is to separate gains from richer spectral information from gains due to spatial modeling. The final benchmark uses:

- official Sen1Floods11 hand-labeled split files
- invalid-pixel masking (`-1` labels excluded from loss/evaluation)
- threshold-tuned spectral baselines
- a 13-band Random Forest baseline
- a corrected 13-band U-Net

## Dataset

This project uses the hand-labeled Sentinel-2 subset of Sen1Floods11.

- Dataset repository: `https://github.com/cloudtostreet/Sen1Floods11`
- Dataset paper: `https://openaccess.thecvf.com/content_CVPRW_2020/papers/w11/Bonafilia_Sen1Floods11_A_Georeferenced_Dataset_to_Train_and_Test_Deep_Learning_CVPRW_2020_paper.pdf`

This repository does **not** redistribute the dataset. Please download Sen1Floods11 directly from the source.
